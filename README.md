# Implentaition of complex JOINS and OPTIMIZATIONS on SimpleDB
## Project consists of:
- Thorough understanding of SimpleDB implementation
- Manual planning of the queries
- Modifying heuristic query planner (Cost calculation)
- Implementaion of join operators (HASH, MERGE-SORT, BLNJ)
- Comparison between CROSS PRODUCT and other JOINS
